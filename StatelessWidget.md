# StatelessWidget

## Text

建立一個樣式化的文字元件

```Dart

import 'package:flutter/material.dart';

void main() {
  runApp(
    Center(
      child: Text(
        'Hello',
        textDirection: TextDirection.ltr,
      ),
    )
  );
}

```
