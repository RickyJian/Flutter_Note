# Widget

讓 APP 在不同的狀態下顯示出不一樣的外觀

## Text

建立一個樣式化的文字元件

```Dart

import 'package:flutter/material.dart';

void main() {
  runApp(
    Center(
      child: Text(
        'Hello',
        textDirection: TextDirection.ltr,
      ),
    )
  );
}

```
